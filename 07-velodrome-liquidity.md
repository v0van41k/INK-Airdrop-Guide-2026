# 07. Velodrome (Liquidity Provision)

**Velodrome** — основной DEX для LP на Superchain (включая Ink).

**Что делать:**
- Добавляй ликвидность в пары (ETH/USDC, INK-related).
- Голосуй за gauges (voting power).
- Получай VELO + bribes.

**Преимущества:** Стабильный доход + сильный сигнал для airdrop.
