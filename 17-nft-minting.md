# 17. Минтинг NFT на Ink

NFT помогают повысить профиль аккаунта.

**Рекомендуемые:**
- **Eye of the Seeker** NFT (если активен)
- **Ink Pass** NFT
- **Templars of the Storm** (с Nado)
- Другие коллекции из раздела Apps → NFT

**Почему важно:** Многие airdrop-программы дают буст за持有 NFT экосистемы.
