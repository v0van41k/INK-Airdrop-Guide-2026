# 03. Бридж активов на Ink

### Лучшие способы
1. **Kraken** → zero-fee withdrawal напрямую на Ink (самый выгодный).
2. Официальный бридж: `https://inkonchain.com/bridge`
3. Сторонние: Across, Stargate, Rhino.fi, Owlto, Brid.gg.

### Рекомендации по сумме
- Минимум: $500–1000 (для нормальной активности).
- Оптимально: $2000–5000+.
- Основные ассеты: ETH, USDC, USDT.

**Совет:** Бриджи сначала маленькую сумму для теста, потом основную.
