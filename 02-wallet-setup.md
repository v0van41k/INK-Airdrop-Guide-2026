# 02. Подготовка кошелька

### Рекомендуемые кошельки
- **Rabby Wallet** (лучший)
- MetaMask
- OKX Wallet

### Добавление сети Ink
- **Network Name**: Ink
- **RPC URL**: `https://rpc-gel.inkonchain.com`
- **Chain ID**: `57073`
- **Currency Symbol**: ETH
- **Block Explorer**: `https://explorer.inkonchain.com`

### Безопасность
1. Используй hardware wallet (Ledger/Trezor) если фармишь крупные суммы.
2. Регулярно проверяй approvals на [revoke.cash](https://revoke.cash).
3. Не оставляй большие суммы на кошельке.
4. Делай отдельный кошелёк под Ink.
