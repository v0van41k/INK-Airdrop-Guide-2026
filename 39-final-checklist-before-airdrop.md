# 39. Финальный чек-лист перед Airdrop

**Перед ожидаемым snapshot (за 1–2 месяца до TGE) убедись, что:**

### ✅ Must-have активности
- [ ] Бридж активов сделан минимум 2–3 раза
- [ ] Регулярные торги на Nado (минимум 4–6 недель)
- [ ] Supply + Borrow на Tydro (долгосрочные позиции)
- [ ] LP на Velodrome (хотя бы 1–2 пары)
- [ ] Mint минимум 2–3 NFT экосистемы
- [ ] Зарегистрирован .ink домен
- [ ] Активность на Kraken Pro (weekly points)
- [ ] Разнообразие транзакций (минимум 8–10 разных dApps)
- [ ] Approvals отозваны
- [ ] Ведётся таблица с историей активности

### ❌ Чего избегать
- [ ] Круглых сумм (1000, 5000 и т.д.)
- [ ] Одинаковых транзакций на нескольких аккаунтах
- [ ] Долгих периодов без активности
- [ ] Только одного типа активности (только LP или только swaps)

**Итоговая оценка:** Если выполнено 90%+ пунктов — шансы на хороший дроп высокие.
