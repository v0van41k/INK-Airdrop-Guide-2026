# 06. Tydro (Lending Protocol)

**Tydro** — основной lending протокол на Ink (на базе Aave V3).

**Стратегии фарминга:**
1. Supply ETH / USDC / USDT.
2. Borrow другие активы (осторожно с liquidation).
3. Держи позиции долго (consistency важна).
4. Реинвестируй награды.

**Ссылка:** Ищи Tydro на [inkonchain.com/apps](https://inkonchain.com/apps)

Это один из самых важных протоколов для eligibility.
